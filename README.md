# Описание
`JScript` для набора номера на телефонах `Tadiran` по **url** схеме `tel` в браузере.
# Использование
В командной строке **Windows** введите следующую команду:
```
wscript caller.min.js <action> [<number>]
```
- `<action>` - Действие которое нужно выполнить.
- `<number>` - Телефонный номер или url для набора.
## Поддерживаемые действия
- `reg` - Зарегистрировать данный скрипт как обработчик протокола `tel`.
- `unreg` - Удалить регистрацию скрипта из обработчиков протокола `tel`.
- `call` - Выполнить набор номера переданного в парамере `<number>`.
# Примеры использования
Зарегистрировать данный скрипт как обработчик протокола `tel`.
```
wscript caller.min.js reg
```
Выполнить набор номера `+71234567890`.
```
wscript caller.min.js call +71234567890
```
